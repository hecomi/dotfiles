names=a visibility=public kind=defined --- a(href = "") -> String --- a(href = ""){ ... } -> String a 要素を生成します。 ブロックを与えると、ブロックを評価した結果が内容になります。 @param href 文字列を指定します。属性をハッシュで指定することもできます。 例: a("http://www.example.com") { "Example" } # => "Example" a("HREF" => "http://www.example.com", "TARGET" => "_top") { "Example" } # => "Example"
